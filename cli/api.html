CLI Reference¶

usage: python3 mc2.py [-h] {upload,run,download,launch,check} ...

Positional Arguments¶

command

Possible choices: upload, run, download, launch, check

Command to run.

Sub-commands:¶

upload¶

Encrypt and upload data.

python3 mc2.py upload [-h] [--sql] [--xgb]

Named Arguments¶

--sql

Encrypt data in Opaque SQL format

Default: False

--xgb

Encrypt data in Secure XGBoost format

Default: False

run¶

Attest the MC2 deployment and run your script.

python3 mc2.py run [-h] [--sql] [--xgb]

Named Arguments¶

--sql

Run the Opaque SQL code specified in the config

Default: False

--xgb

Run the Secure XGBoost code specified in the config

Default: False

download¶

Download and decrypt results from your computation.

python3 mc2.py download [-h] [--sql] [--xgb]

Named Arguments¶

--sql

Decrypt data in Opaque SQL format

Default: False

--xgb

Decrypt data in Secure XGBoost format

Default: False

launch¶

Launch compute service using commands specified in config

python3 mc2.py launch [-h] [--sql] [--xgb]

Named Arguments¶

--sql

Launch Opaque SQL service

Default: False

--xgb

Launch Secure XGBoost service

Default: False

check¶

Check status of compute service

python3 mc2.py check [-h]
